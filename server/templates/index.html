Lamport Clock

Logical
Loại Scalar (1 số nguyên)
Happened-before ✓ Hỗ trợ (1 chiều)
Concurrent detection ✗ Không phát hiện
Kích thước O(1)
Quy tắc Send LC += 1; send(LC)
Quy tắc Receive LC = max(LC, recv_LC) + 1
Ưu điểm Đơn giản, overhead thấp
Nhược điểm Không phân biệt được concurrent events

Vector Clock

Logical
Loại Vector (mảng N phần tử)
Happened-before ✓ Hỗ trợ (đầy đủ)
Concurrent detection ✓ Phát hiện được
Kích thước O(N) — N = số node
Quy tắc Send VC[self]++; send(VC)
Quy tắc Receive VC[i]=max(VC[i],recv[i])∀i; VC[self]++
Ưu điểm Xác định hoàn toàn thứ tự nhân quả
Nhược điểm Overhead tăng theo số node O(N)

NTP (Network Time Protocol)

Physical
Loại đồng hồ Đồng hồ VẬT LÝ (Wall Clock)
Mục đích Đồng bộ thời gian thực giữa các máy
Công thức Delay δ = ((T2-T1) + (T4-T3)) / 2
Công thức Offset θ = ((T2-T1) - (T4-T3)) / 2
